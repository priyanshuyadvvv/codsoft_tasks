# codsoft_tasks
# Python Mini Projects

This repository contains three beginner-friendly Python projects developed using Python.

## Projects

### 1. To-Do List
A simple command-line application to manage daily tasks.

#### Features
- Add tasks
- View tasks
- Remove tasks
- Exit the application

#### Technologies
- Python

---

### 2. Simple Calculator
A basic calculator that performs arithmetic operations.

#### Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero

#### Technologies
- Python

---

### 3. Password Generator
A random password generator using Python.

#### Features
- User chooses password length
- Generates strong random passwords
- Uses letters, numbers, and special characters

#### Technologies
- Python
- random module
- string module

---

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/your-username/python-mini-projects.git
```

2. Open the project folder:
```bash
cd python-mini-projects
```

3. Run any project:
```bash
python task1.py
python task2.py
python task3.py
```
